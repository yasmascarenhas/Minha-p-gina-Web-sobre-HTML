Aprendendo HTML

HTML é uma linguagem para criar página Web

  • Head é o que o usuário não vê, como codificação do texto, título da aba do navegador
  • Body é o que o usuário vê como título, paragráfos
  • Html abre e fecha o código
  • HTML é uma linguagem de marcação.
  • O HTML é a "espinha dorsal" da internet e fundamental para qualquer desenvolvedor web.
